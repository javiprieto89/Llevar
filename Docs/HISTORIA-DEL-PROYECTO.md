# Disclaimer

## Sobre este Proyecto

Este programa tiene un significado especial para mí. Con 12 años, aprendí a programar en Batch/BAT/DOS gracias a este mismo proyecto y a una persona que tuvo muchísima paciencia conmigo en su momento.

Años después, me pareció adecuado llevar ese viejo script Batch al mundo moderno como una forma de aprender a programar en PowerShell. Este proyecto no solo es una modernización técnica, sino también un tributo a aquellos primeros pasos en la programación y a la persona que me guió en ese camino.

## Propósito Educativo

Al igual que el script original me enseñó los fundamentos de la programación cuando era niño, esta versión en PowerShell representa un nuevo viaje de aprendizaje. Cada módulo, cada función, cada mejora implementada ha sido una oportunidad para comprender mejor PowerShell y las prácticas modernas de desarrollo.
